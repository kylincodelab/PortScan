# PortScan
一、中文版说明

1、作者：lazynms

2、用法：

(进入命令行（cmd或terminal）：切换到当前文件夹，运行 “python PortScan.py -H yourhost -p 21,22,80,443,8080,6792”)

其中，youhost为要扫描的主机域名或主机IP，-p后接要扫描的端口，以逗号分隔
